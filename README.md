## 🚀 Featured Projects

### 🌳 [Tree_Of_Thought](https://github.com/Cerynitius/Tree_Of_Thought)
An external Tree-of-Thought reasoning system with a **FastAPI** backend, **browser UI**, and FSM-governed node lifecycle. Turns opaque model completions into an inspectable, steerable reasoning tree with scoring, pruning, and **SymPy**-backed symbolic computation.

> **Stack:** Python · FastAPI · SymPy · FSM · Multi-model routing

---

### 🤝 [agent_team](https://github.com/Cerynitius/agent_team)
A 9-agent autonomous orchestration framework built on a shared task board with claim-based DAG scheduling. Features a Coder–Reviewer loop, dynamic prompt evolution, mailbox messaging, and **Redis / PostgreSQL** session persistence.

> **Stack:** Python · Pydantic · Redis · PostgreSQL · LM Studio (Qwen)

---

### ⚖️ [court](https://github.com/Cerynitius/court)
A multi-agent courtroom trial simulation with 16 AI agents (judge, attorneys, witnesses, 8 jurors). Runs full trials end-to-end and auto-generates **DPO training datasets** for preference alignment fine-tuning.

> **Stack:** Python · LM Studio (Qwen) · DPO · Multi-agent FSM

---

### 🎮 [game_project](https://github.com/Cerynitius/game_project)
A 2D top-down survival game with procedurally generated terrain, melee/ranged combat, sanity mechanics, and a live **GPT-4o-mini** world director narrating the game state in real time.

> **Stack:** Python · Pygame · OpenAI API (GPT-4o-mini)

---

### 🔬 [llmTrain](https://github.com/Cerynitius/llmTrain)
A minimal end-to-end pipeline for training a BPE tokenizer and a Transformer language model from scratch. Trained on an NVIDIA A800 GPU; model published on **HuggingFace** as [MiniLLM-0.1B](https://huggingface.co/Hippocrene/MiniLLM-0.1B).

> **Stack:** Python · PyTorch · HuggingFace · BPE Tokenizer · Transformer
